## express + wxapp 

### 使用 
#### serveer
`````
提前安装node
cd serve 
npm install 
npm start
`````

#### wxapp
``````
提前安装taro
cd wxapp
npm install
# watch
taro build --type weapp --watch
# build
taro build --type weapp
``````

#### 修改admin
`````
cd admin-veiws
npm install
# 修改
npm start
# 打包
npm run-script build
`````
